אודות הקורס

קורס זה נועד להקנות לסטודנטים ידע בסיסי בתחום מערכות מידע וניהול נתונים. נלמד על סוגי מערכות מידע, תהליכי ניהול נתונים, טכנולוגיות אחסון וגישה, ואת ההשפעה של מידע על קבלת החלטות.

נושאי הקורס

  • מערכות מידע: סוגים, ארכיטקטורות ותפקידים
  • ניהול נתונים: איסוף, אחסון, עיבוד ושמירה
  • בסיסי נתונים: SQL, MySQL, PostgreSQL
  • Big Data: Hadoop, Spark, Cloud Computing
  • בינה מלאכותית וניתוח נתונים: Machine Learning, Data Mining
  • אבטחת מידע ופרטיות

למי מתאים?

הקורס מתאים לסטודנטים, אנשי מקצוע בתחומים שונים, ולכל מי שמעוניין להבין את העולם של מידע ונתונים.

תכנים מתקדמים

* ניתוח נתונים מתקדם עם Python (Pandas, NumPy) * פיתוח יישומי BI (Tableau, Power BI) * Data Warehousing ו-Data Modeling * ניתוח נתונים גדולים (Big Data)
